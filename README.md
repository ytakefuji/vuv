# vuv
vuv is to evaluate the vaccine effects on mortality by age group 
between fully vaccinated and unvaccinated. 
Five age groups include 18-29,30-49,50-64,65-79, and 80+. 

To install vuv, 

$ pip install vuv

To run vuv, enter one of five age groups.

$ vuv 80+
